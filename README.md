## ğŸš€ Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (>= 14.x)
- **Yarn** (or npm)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/superbabii/React-Poker-Game.git
   cd React-Poker-Game
   ```

2. Install dependencies:
   ```bash
   yarn install
   # Or, using npm
   npm install
   ```

3. Start the development server:
   ```bash
   yarn start
   # Or, using npm
   npm start
   ```
